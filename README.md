# Bloom English A1

React/Vite version of the Bloom English A1 learning platform.

## Run locally

```bash
npm install
npm run dev
```

## Publish with Vercel

1. Upload this folder to GitHub.
2. Go to Vercel.
3. Import the GitHub repository.
4. Deploy.

## Notes

- Microphone/speech recognition works best in Chrome or Edge and requires HTTPS or localhost.
- AI Tutor includes a local fallback so the app works without an API key.
- If you want real Gemini API responses, connect the API through a secure backend/proxy instead of exposing the key in frontend code.
